# Rust AUTD Examples

## One AUTD test

```
cargo run --bin soem --features all    
```

or if you use TwinCAT,
```
cargo run --bin twincat --features all    
```

## Two AUTD test

```
cargo run --bin two_autd --features all    
```

# Caution

If you use Linux/macOS, you may have to run as root.
